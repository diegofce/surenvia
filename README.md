# Sur Envía

Landing page de Sur Envía, una web promocional para servicios de mensajería, domicilios, radicación de documentos y entregas para empresas y comercios.

## Tecnologias

- React 19
- Vite
- Framer Motion
- Lucide React
- Tailwind CSS por CDN en `index.html`

## Funcionalidades

- Hero principal con llamado a WhatsApp.
- Botones de contacto directo por WhatsApp y telefono.
- Secciones de servicios, diferenciales, cobertura y testimonios.
- Menu responsive con navegacion interna.
- Logo principal que retorna al inicio de la pagina.

## Requisitos

- Node.js 18 o superior.
- npm.

## Instalacion

```bash
npm install
```

## Scripts disponibles

```bash
npm run dev
npm run build
npm run preview
```

## Uso local

1. Instala dependencias con `npm install`.
2. Ejecuta `npm run dev`.
3. Abre la URL que muestre Vite, normalmente `http://localhost:5173/` o el puerto libre que asigne.

## Estructura principal

- `src/App.jsx`: componente principal de la landing.
- `src/main.jsx`: punto de entrada de React.
- `src/styles.css`: estilos globales.
- `index.html`: plantilla principal y metadata.
- `vite.config.js`: configuracion de Vite con React.
